# Predicting Customer Churn in a Subscription-Based Business

## Project Overview
Customer churn prediction involves analyzing customer data to determine the likelihood of customers discontinuing their subscription services. This project aims to understand the key factors that contribute to customer churn and employs various data analysis and modeling techniques to predict potential churners, enabling businesses to implement strategies to improve customer retention.

## Report Summary
This project is documented in the report titled "Soen471-G28_A1_Report.pdf". The report provides an in-depth analysis of the methodologies used, the results obtained from various predictive models, and recommendations for improving customer retention strategies based on the findings.

## Project Description
### Objectives
- To identify the key factors influencing customer churn in subscription-based businesses.
- To develop predictive models that accurately forecast customer churn.
- To provide actionable insights and recommendations for retention strategies.

### Methodology
The project employs data preprocessing, exploratory data analysis (EDA), feature selection, and the application of various machine learning algorithms to predict customer churn. Techniques such as logistic regression, decision trees, and ensemble methods are evaluated for performance accuracy.

### Results
The results indicate significant predictive variables for customer churn and the comparative performance of models. Insights derived from the data aid in understanding customer behavior better, enabling businesses to tailor their strategies effectively.
